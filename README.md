# Plateforme Objets Perdus

Plateforme web pour signaler et retrouver des objets perdus.

## 🚀 Installation

\`\`\`bash
npm install
cp .env.example .env
npm start
\`\`\`

## 🏗️ Architecture
- **Backend** : Express.js
- **Database** : SQLite
- **Frontend** : EJS/HTML

## 📋 Fonctionnalités
- Authentification utilisateur
- Signalement d'objets perdus
- Profil utilisateur
- Recherche d'objets

## 🔒 Sécurité
- Sessions sécurisées
- Validation des inputs
- Protection CSRF (à ajouter)
